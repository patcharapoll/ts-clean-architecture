# ts-clean-architecture

## Install

- Clone the project
- Install dependencies:
  - Using yarn
  ```bash
  $ yarn install
  ```

  - Using npm
  ```bash
  $ npm install
  ```

## Running

- Running the script

  This will run a single node script file and will execute pre-defined methods as an example.

    - Using yarn
    ```bash
    $ yarn start
    ```
    - Using npm
    ```bash
    $ npm run start
    ```
